# Open Router Integration Fix & Free Tier Implementation

## Context

The horary astrology app currently has a multi-provider LLM system supporting Ollama (local), OpenRouter (cloud with 290+ models), and Anthropic. However, **Open Router is not working in the deployed app** because:

1. **Legacy code is still active**: The main reading generation functions in `/src/utils/llm.ts` are still using the OLD Ollama-only client implementation, not the NEW multi-provider system in `/src/utils/llm/client.ts`
2. **No free tier for new users**: Users must provide their own API key immediately, creating friction for first-time users

The user wants to:
1. Fix Open Router to work properly in deployment by migrating to the new multi-provider system
2. Allow new users to try the app with FREE Open Router models using the owner's API key, with usage limits:
   - **Token-based limit**: Track actual token consumption (more fair than request count)
   - **Time-based rate limit**: 3 requests per hour to prevent abuse
   - After limit: Prompt users to add their own API key
3. Implement using **Netlify serverless function** to keep API key secure server-side, with localStorage tracking for rate limiting

#### 2.3: Create Netlify Serverless Function for Free Tier

**New file: `/netlify/functions/llm-proxy.ts`**

Create a serverless function that proxies LLM requests using the owner's API key:

```typescript
import type { Handler, HandlerEvent } from '@netlify/functions';

const OPENROUTER_API_KEY = process.env.OPENROUTER_FREE_TIER_KEY;

export const handler: Handler = async (event: HandlerEvent) => {
  // Only allow POST requests
  if (event.httpMethod !== 'POST') {
    return {
      statusCode: 405,
      body: JSON.stringify({ error: 'Method not allowed' }),
    };
  }

  // Only allow requests from your domain (CORS)
  const origin = event.headers.origin || '';
  const allowedOrigins = [
    'http://localhost:5173',
    'https://your-app.netlify.app', // Update with actual domain
  ];

  if (!allowedOrigins.includes(origin)) {
    return {
      statusCode: 403,
      body: JSON.stringify({ error: 'Forbidden' }),
    };
  }

  try {
    const requestBody = JSON.parse(event.body || '{}');

    // Forward request to OpenRouter
    const response = await fetch('https://openrouter.ai/api/v1/chat/completions', {
      method: 'POST',
      headers: {
        'Content-Type': 'application/json',
        'Authorization': `Bearer ${OPENROUTER_API_KEY}`,
        'HTTP-Referer': origin,
        'X-Title': 'Horary Astrology App',
      },
      body: JSON.stringify(requestBody),
    });

    const data = await response.json();

    return {
      statusCode: response.status,
      headers: {
        'Access-Control-Allow-Origin': origin,
        'Access-Control-Allow-Headers': 'Content-Type',
        'Content-Type': 'application/json',
      },
      body: JSON.stringify(data),
    };
  } catch (error: any) {
    console.error('Proxy error:', error);
    return {
      statusCode: 500,
      body: JSON.stringify({ error: 'Internal server error' }),
    };
  }
};
```

      }
    }

    const openai = createLLMClient();
    const model = getCurrentModel();
